ES Modules
ES Modules (ESM) are the modern JavaScript module system introduced in ES6.
They let you split your code into reusable pieces using:
Import:
import { add } from ‘./math.js’;
Export
Export function add(a,b) {
return a+b;
}
Key Features
Native browser support
Static analysis (fast bundling, tree-shaking)
Works in node.js with .mjs or “type” : “module”
npm(Node Package Manager) is:
The package manger that comes with Node.js
Used to install and manage JavaScript libraries
Use a package.json file to track dependencies
Ex.
npm install lodash
Then use it in your code(if you’re using ESM):
import_from ‘lodash’;
How ES Module and npm Work Together
Node.js now support ES Modules, so you can use ESM when writing code that uses npm packages.
Step 1: set “type” : “module” in package.json
{
“type”: “module”
}
Step 2: install a package
npm install axios
step 3: import it using ESM
import axios from “axios”;
const res = await axios.get(‘https: //example.com’);
console.log(res.data);
Example project using ESM+npm
My project/
|- package.json
|- index.js
|-utils.js
Package.json
{
“name” : “my-project”,
“type” : “module”
“dependencies” : {}
}
util.js
export function greet(name) {
   return `Hello, ${name}!`;
}
index.js
import { greet } from ‘./utils.js’;
console.log(greet(“Satish”));
Run:
node index.js
