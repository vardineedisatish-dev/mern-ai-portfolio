- Definition: Containers used to store values.
- Ways to declare:
  - const â†’ value cannot be reassigned.
  - let â†’ value can be reassigned later.
  - var â†’ older way; function-scoped, hoisting issues.

Hoisting
- Declarations are moved to the top during compilation.
- let and const are hoisted but not initialized â†’ Temporal Dead Zone (TDZ).

Example:

let x = 5;

console.log(y); // undefined (var is initialized to undefined)
var y = 10;

Naming Rules
- Must start with a letter, _, or $.
- No spaces.
- Use camelCase convention â†’ totalAmount.

ðŸ”¹ Operators

1. Arithmetic
+, -, *, /, %, **
Example:

2. Assignment
=, +=, -=, *=, /=
Example:

3. Comparison
==, ===, !=, !==, >, <, >=, <=
Examples:

4. Logical

Examples:

6. Unary & Ternary

Unary:
++, --, typeof
Examples:
let x = 5;
x++; // 6

let y = 10;
y--; // 9

typeof "hello"; // "string"

Ternary:
Shortcut for if...else
Syntax: condition ? valueIfTrue : valueIfFalse

Example:
let age = 18;
let msg = age >= 18 ? "Adult" : "Minor";
console.log(msg); // "Adult"

ðŸ”¹ Booleans, Truthy/Falsy & Decisions

Falsy values: false, 0, "", null, undefined, NaN
Truthy values: everything else (even "0", "false", [], {})

Logical Operators Returning Values:

"" || "default";    // "default" (OR returns first truthy)
null ?? "fallback"; // "fallback" (nullish coalescing)
